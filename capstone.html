Using Machine Learning to Predict Failures in a Particle Accelerator

I am currently working with Colin O'Brien, Ryan Pindale, and Timur Gulur for the Jefferson Lab under the Department of Energy. Using sensor data from the machine provided by Jefferson Lab, our goal is to build a model that can predict whether or not the machine is about to fail. Many of these machine failures cause extended downtime and/or damage to the machine itself, so being able to preemptively identify these failures in real time could potentially save Jefferson Lab tens of thousands of dollars. We are exploring a range of models including few shot learning, Bayesian neural networks, and Gaussian processes to determine which one best identifies these machine failures preemptively.
